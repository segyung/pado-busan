# PADO GitHub 업로드 안내

이 파일은 GitHub/Netlify 연결 배포용 최종 프로젝트입니다.

## 반영 내용
- 상단 헤더에 로그인/회원가입 버튼이 항상 보이도록 수정
- 로그인 후 사용자 메뉴/마이페이지/로그아웃으로 전환
- 공지 메뉴 추가
- 공지사항에는 활동 점수, 밋업 개설, 운영 수칙, 기수제 안내만 표시
- 오늘의 번개 관련 공지 내용 제외
- 오늘의 번개 자동 생성/자동 전환 문구 제거
- 오늘의 번개는 운영진이 수동 등록
- 밋업 만들기 버튼은 바로 생성 화면으로 이동
- GitHub/Netlify 빌드 확인 완료

## Netlify 설정
Build command: npm run build
Publish directory: dist
